# AQUA-Water-Quality-Inteligence
AQUA is an end-to-end Machine Learning solution developed to advance UN Sustainable Development Goal 6 (Clean Water and Sanitation). By leveraging predictive analytics, AQUA provides an automated system for assessing water potability, helping communities and organizations identify safety risks before they become health crises.The core of the system is a Random Forest Classifier trained on complex chemical markers. Because real-world environmental sensor data is often inconsistent, the project features a custom-built data cleaning pipeline designed to handle the high "noise" and missing values typical of field-deployed sensors.Key FeaturesPredictive Analytics: Analyzes 9 critical water parameters (pH, Sulfate, Chloramines, etc.) to determine if water is safe for human consumption.Robust Data Cleaning: Implements iterative median imputation and feature scaling ($StandardScaler$) to handle the unit misalignment and sensor failure gaps found in the water_potability.csv dataset.Interactive Dashboard: A colorful Streamlit frontend that allows users to perform "What-If" analysis by manually adjusting chemical levels using a UI styled with a sleek #0E1116 charcoal theme.Explainable AI: Uses feature importance mapping to highlight that Sulfate levels and pH balance are the primary drivers of safety, aligning the model's logic with WHO standards.Tech StackLanguage: PythonLibraries: Scikit-Learn, Pandas, NumPyVisualization: Matplotlib, Seaborn, StreamlitDomain: Environmental Data Science / SDG 6
